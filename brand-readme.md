# Automate My ☕ — Brand Guidelines

> One rule before anything else: if it sounds like another tech company, rewrite it.

---

## Logo

**Concept:** "AU[☕]MATE" — the coffee cup replaces the letter O in AUTOMATE. "my" sits below in handwritten script.

**Usage:**
- Always on cream (`#FAF7F2`) or white (`#FFFFFF`) backgrounds
- Never stretch, rotate, or recolor
- Minimum clear space: equal to the height of the cup icon on all sides
- Don't place on busy backgrounds or dark backgrounds (unless you have a white version)

---

## Colors

| Name | Hex | Use for |
|---|---|---|
| Terracotta Orange | `#BE582A` | Logo, headlines, buttons, accents — the hero color |
| Warm Cream | `#FAF7F2` | Page backgrounds, cards |
| Near-Black | `#1C1917` | Body text, dark UI elements |
| Pure White | `#FFFFFF` | Text on orange, clean space |

**Rules:**
- Never use more than these 4 colors
- Orange on cream = hero combo
- Black text on cream = body copy
- White text on orange = buttons/CTAs
- Never orange text on white (too low contrast)

---

## Typography

| Role | Font | Use for |
|---|---|---|
| Display | **NationalPark Bold** | Headlines, logo wordmark, section titles |
| Accent | **NothingYouCouldDo** | Script touches only (like the "my" in the logo) — use sparingly |
| Body | **Outfit Regular** | Paragraphs, descriptions, UI text |
| Mono (optional) | **GeistMono Regular** | Code, technical snippets |

**Size scale:**
- Hero: 64–80px
- Section heading: 40–48px
- Subheading: 24–28px
- Body: 16–18px
- Caption/label: 12–14px
- Button: 16px, bold, uppercase

## Tone of Voice

**We are:** Energetic. Playful. Direct. Human.

**We are not:** Corporate. Technical. Buzzword-y. Boring.

**Write like this:**
- Short sentences. One idea at a time.
- Active voice always.
- Talk to one person, not a crowd.
- Lead with the benefit, not the feature.

**Word swaps:**

| ❌ Never say | ✅ Say instead |
|---|---|
| AI-powered | automation that just works |
| Leverage | use |
| Streamline your workflow | stop doing the same thing twice |
| Cutting-edge | actually smart |
| Seamless integration | it just connects |
| Enterprise-grade | built for real work |
| Innovative solution | a better way |

## UI Copy & Microcopy

**Buttons:**
- Get Started → `Let's go →`
- Contact → `Talk to us`
- Learn More → `Show me`
- Submit → `I'm in`
- Book a call → `Let's chat ☕`

**Section labels:**
- Services → `What we do`
- Portfolio → `Stuff we've built`
- About → `Who's behind this`
- Pricing → `Pick your plan`
- Contact → `Got a boring task?`

**States:**
- Loading → `Brewing…`
- Empty state → `Nothing here yet. Go grab a coffee.`
- Form sent → `Got it. We'll be in touch faster than your morning brew.`
- Error → `Something went sideways. Try again?`

**Coming soon page:**
- Headline: `Something's brewing.`
- Subline: `We're automating the boring stuff so you don't have to.`
- Email label: `Drop your email, we'll handle the rest.`
- CTA button: `Notify me →`

---

## Do's & Don'ts

**Always:**
- Lead with the benefit, not the feature
- Use short, punchy sentences — fragments are fine
- Let the orange do the heavy lifting visually
- Keep lots of white/cream breathing room
- Use the cup icon playfully but intentionally

**Never:**
- Use gradients or drop shadows
- Add more than one accent color
- Write sentences longer than 20 words in UI copy
- Use more than 2 fonts on any page
- Sound like an AI company
